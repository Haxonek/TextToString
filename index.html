Convert text to a string!

Converts a file, long string, etc that contains returns, quotation marks (") and tabs into a string that's safe to enter into code while not losing the formatting of the string.

Check the checkboxes depending on what you wanted added/removed from the file, and it'll format it into a string that can be entered into code at the bottom.
